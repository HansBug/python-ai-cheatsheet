# 任意多边形面积公式 面试攻略

## 这是什么？

这是计算几何里最基础、也最值得面试时讲清楚的一条公式。

如果只用一句话来讲：

> 对按顺序给出的简单多边形顶点，面积可以直接写成相邻顶点叉积和的一半，也就是经典的 Shoelace Formula。

它的重要性在于：

- 实现非常短
- 推导很漂亮
- 和叉积、三角形有向面积、凸包、点在多边形内等很多几何问题都能连起来

## 核心机制

### 1. 我们到底在算什么面积？

设简单多边形顶点按顺序给出：

$$
P_0, P_1, \ldots, P_{n-1}, \quad P_i = (x_i, y_i)
$$

这里默认：

- 顶点顺序沿边界走一圈
- 多边形是简单多边形，也就是边不自交

如果顶点是逆时针，公式会给出正面积；如果是顺时针，公式会给出负面积。  
所以更严格地说，我们先算的是**有向面积**。

### 2. 从代码先看最终公式

如果直接对着实现看，最短代码就是：

```python
def signed_polygon_area(points):
    total = 0.0
    n = len(points)
    for i in range(n):
        x1, y1 = points[i]
        x2, y2 = points[(i + 1) % n]
        total += x1 * y2 - y1 * x2
    return 0.5 * total
```

这段代码每一部分都对应一个明确数学对象：

- `points[i]` 和 `points[(i + 1) % n]`：相邻的两个顶点
- `x1 * y2 - y1 * x2`：二维叉积，也就是从原点出发形成的平行四边形有向面积
- `0.5 * total`：平行四边形面积的一半，就是三角形面积

所以整条公式其实就是：

$$
S_{\mathrm{signed}}
=
\frac{1}{2}
\sum_{i=0}^{n-1}
\left(x_i y_{i+1} - y_i x_{i+1}\right)
$$

其中下标按模 `n` 处理，也就是 `P_n = P_0`。

取绝对值后就是通常说的几何面积：

$$
S = \left| S_{\mathrm{signed}} \right|
$$

### 3. 先证明一个更小的结论：三角形有向面积

对两个向量：

$$
a = (x_1, y_1), \quad b = (x_2, y_2)
$$

它们的二维叉积写成：

$$
a \times b = x_1 y_2 - y_1 x_2
$$

它的几何意义是：

- 绝对值等于由 `a` 和 `b` 张成的平行四边形面积
- 一半就是三角形 `\triangle OAB` 的面积
- 正负号反映方向：逆时针为正，顺时针为负

所以三角形 `\triangle OAB` 的有向面积是：

$$
S_{\mathrm{signed}}(\triangle OAB)
=
\frac{1}{2}(a \times b)
=
\frac{1}{2}(x_1 y_2 - y_1 x_2)
$$

这就是后面整个多边形公式的基础。

### 4. 多边形面积为什么能写成一圈叉积和？

核心想法很简单：

> 把多边形看成一串相邻边和原点组成的三角形的有向面积之和。

如果把每条边 `P_i -> P_{i+1}` 都和原点 `O` 组成一个三角形，那么第 `i` 项贡献就是：

$$
S_i
=
\frac{1}{2}
\left(
x_i y_{i+1} - y_i x_{i+1}
\right)
$$

把所有边加起来，就得到：

$$
S_{\mathrm{signed}}
=
\sum_{i=0}^{n-1} S_i
=
\frac{1}{2}
\sum_{i=0}^{n-1}
\left(
x_i y_{i+1} - y_i x_{i+1}
\right)
$$

### 5. 这个推导为什么对任意简单多边形都成立？

这是最容易被追问的地方。

如果多边形是凸的，这件事直觉很好理解：  
这些三角形会刚好拼出整个多边形。

如果多边形是凹的，为什么也成立？

关键在于我们算的是**有向面积**。

也就是说：

- 落在多边形内部的部分按正负方向累加
- 落在外面的那些“多算”区域，会在别的边对应三角形里以相反方向被减掉

所以最终留下来的，恰好就是多边形本身的有向面积。

更正式一点，可以从“参考点任意”来证明。

### 6. 为什么参考点取原点不会出错？

你可能会问：  
如果原点根本不在多边形里，为什么还能这样算？

这是因为公式其实和参考点无关。

假设不用原点，而是任取一个参考点 `Q`，那么每一项三角形面积会变成：

$$
\frac{1}{2}
\left(
(P_i - Q) \times (P_{i+1} - Q)
\right)
$$

把它展开：

$$
(P_i - Q) \times (P_{i+1} - Q)
=
P_i \times P_{i+1}
- P_i \times Q
- Q \times P_{i+1}
$$

又因为叉积反对称：

$$
- P_i \times Q = Q \times P_i
$$

所以求和后，中间项会变成：

$$
\sum_{i=0}^{n-1} Q \times (P_i - P_{i+1})
$$

而顶点首尾相接，所以：

$$
\sum_{i=0}^{n-1} (P_i - P_{i+1}) = 0
$$

于是这些额外项全部消掉，只剩：

$$
\sum_{i=0}^{n-1} P_i \times P_{i+1}
$$

这就证明了：

> 不管参考点选在哪里，最后得到的总有向面积都一样。

所以把参考点直接取成原点完全没问题。

### 7. Shoelace Formula 为什么叫鞋带公式？

把式子写展开：

$$
S_{\mathrm{signed}}
=
\frac{1}{2}
\left[
(x_0 y_1 + x_1 y_2 + \cdots + x_{n-1} y_0)
-
(y_0 x_1 + y_1 x_2 + \cdots + y_{n-1} x_0)
\right]
$$

如果把 `x` 列和 `y` 列上下排开，交叉相乘的连线看起来像鞋带，所以叫 Shoelace Formula。

## 面试高频问题

### 1. 这个公式对凹多边形也成立吗？

对简单凹多边形成立。因为它本质上算的是有向面积，外部多算部分会相互抵消。

### 2. 顶点顺序反过来会怎样？

面积符号会反过来：逆时针为正，顺时针为负。取绝对值后几何面积不变。

### 3. 这个公式对自交多边形成立吗？

它给出的会是代数面积，不一定等于通常意义下的几何填充面积。所以这里默认简单多边形。

### 4. 为什么代码里要写 `(i + 1) % n`？

因为最后一个顶点要和第一个顶点闭环相连。

## 最小实现

完整代码最适合边看边讲：

```python
def signed_polygon_area(points):
    total = 0.0
    n = len(points)
    for i in range(n):
        x1, y1 = points[i]
        x2, y2 = points[(i + 1) % n]
        total += x1 * y2 - y1 * x2
    return 0.5 * total


def polygon_area(points):
    return abs(signed_polygon_area(points))
```

这里的运行逻辑就是：

1. 沿边界顺序枚举每一条边
2. 把每条边对应成一项叉积
3. 把所有叉积加起来
4. 最后乘 `0.5`

## 工程关注点

### 1. 一定要明确输入前提

最小实现默认输入是简单多边形，且顶点已经按边界顺序给出。

### 2. 浮点误差要有心理准备

如果顶点坐标是浮点数，结果通常也会有正常的数值误差。

### 3. 自交多边形不能直接按普通面积理解

这时公式返回的是代数面积，不是“填色区域的面积”。

## 常见坑点

### 1. 顶点顺序乱了

如果点集不是沿边界顺序给的，公式直接失效。

### 2. 忘了首尾闭环

最后一项必须包含 `P_{n-1}` 到 `P_0`。

### 3. 把有向面积和几何面积混为一谈

符号本身是有意义的，别一上来就只记绝对值。

## 面试时怎么讲

如果面试官让你介绍任意多边形面积公式，可以按这个顺序讲：

1. 先说这是 Shoelace Formula，本质上是相邻顶点叉积和的一半
2. 每一项对应一条边和参考点形成的三角形有向面积
3. 把所有边的三角形面积加起来，就得到整个多边形的有向面积
4. 这个推导对简单凹多边形也成立，因为外部多算部分会按方向抵消
5. 最后取绝对值就是通常说的几何面积

一个简洁版本可以直接讲：

> 任意简单多边形的面积可以写成相邻顶点叉积和的一半，也就是 Shoelace Formula：`S = 1/2 * |sum(x_i y_{i+1} - y_i x_{i+1})|`。它的本质是把多边形边界拆成一圈和参考点组成的三角形有向面积之和。逆时针顶点顺序得到正值，顺时针得到负值，取绝对值后就是几何面积。
